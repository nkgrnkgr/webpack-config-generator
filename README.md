Webpack4 Config Generator
----

Generate a example of [webpack](https://webpack.js.org/) config file by selecting some options.

https://nkgrnkgr.github.io/webpack-config-generator/

### Running local


Install packages.

```bash
$ yarn
```

```bash
$ npm install
```

Start webpack-dev-server.

```bash
$ yarn start
```
```bash
$ npm run start
```
